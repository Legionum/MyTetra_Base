Перемещение объектов плавное (или с анимацией)


  1.  Создать новый blueprint класса actor (например, sphere)
  2. Создать программу (используется нод Simple Move To Location)


  1. Добавить на уровень объект NavMeshBoundsVolume (задает разрешенную для перемещений персонажей зону, без него персонаж не будет двигаться даже при условии правильной настройки всех остальных параметров). Объект растянуть на весь уровень.
