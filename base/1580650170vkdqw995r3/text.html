Команды в консоли:


Список установленных драйверов для графики

mhwd  -li


1) Удалить драйвер Nvidia (может быть установлен другой драйвер)


sudo mhwd -r pci video-hybrid-intel-nvidia-418xx-bumblebee


2) Удалить bumblebee (программу для переключения с интегрированной видеокарты на дискретную)


sudo pacman -Rс bumblebee

3) Установить новый драйвер (версии 435...)


sudo mhwd -i pci video-nvidia-435xx


4) Установить пакет prime (автор @Feresey)

В Manjaro можно установить через графическую программу установки пакетов и других программ


После этого можно запускать игры.
