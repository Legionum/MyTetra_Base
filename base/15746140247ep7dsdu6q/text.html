Общие примечания


Blender 2.8-----> Работа с объектом (изделием, персонажем)---> Режим объекта --->Режим редактирования---> Экструдирование (вытягивание). Выделить вершины и нажать "E", после этого вытягивать в нужную сторону.

 

Объединить несколько объектов в один

- Выделить все нужные объекты (поочерёдно нажимать на объекты удерживая клавишу "Shift");

- Нажать "Cntrl + J"

 

Установить центр объекта

ПКМ на объекте --->Задать центральную точку --->  

Обязательные аддоны в Blender

- Rigging: Rigify (автоматически создает скелет человека для дальнейших анимаций и поз)

 

Экспорт в форматы fbx и obj:

Можно взять исходную модель и поставить ее в положение T-Pose.

 - Для этого нужно создать примитивный скелет человека (можно с помощью аддона Rigging:Regify);

- Находясь в режиме поз, поставить кости на нужное место (положение);

- Экспортировать модель в формате obj для создания анимаций на сайте Mixamo.
