Можно было и про Линукс пару слов написать. На кампе с Ubuntu ttl по умолчанию также 64 как и на Андроидах и айосах. Но чтоб на нем сменить ttl на 65 не нужно не рут не чего-то еще. просто в терменале набрать

sudo iptables -t mangle -A POSTROUTING -j TTL --ttl-set 65

и все. Разумеется что можно использовать это и в в виде скрипта. У меня этот скрипт лежит по пути ~/Tools/yota.sh после сохранения дал права на запуск.

sudo chmod +x ./Tools/yota.sh

при использовании винды с помощью свистков написано много, а вот про мой случай я так и не видел ни одного поста. видимо таких извращенцев мало


создай текстовый документ с содержимым

Windows Registry Editor Version 5.00


[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\Tcpip\Parameters]

"DefaultTTL"=dword:00000041


присвой файлу расширение reg вместо txt

тыкни на файле и проведи слияние с реестром. возможно в твоем случае вместо TTL=65 тебе нужно выставить TTL=64 для этого поменяй в создаваемом reg файле вместо 00000041 поставь 00000040. в общем суть проблемы YOTA не делает обрезание только если TTL=64 с первого устройства, при каждом последующем сетевом устройстве TTL уменьшается на еденицу. Если у тебя планшет не имеет виртуальных сетевых карт то должен быть TTL=64, если при трасеровке видно что до модема TTL уже уменьшился на определенное количество шагов то нужно увеличевать TTL на это количество.

Инструкция для Linux (Ubuntu)


Суть в установке и запуске специального скрипта NIPE, который делает сеть Tor Network шлюзом по умолчанию.


Важное примечание: приложение Yota должно быть удалено с телефона.


1. Установка Perl


1.1 Обновление Ubuntu


sudo apt-get update -y

sudo apt-get upgrade -y


1.2 Установка Perl

sudo apt-get install perl -y


1.3 Установка модуль для установки других модулей через CPAN:

sudo apt-get install build-essential


1.4 Обновить модуль:

sudo perl -MCPAN -e 'install Bundle::CPAN'


1.5 Установка интерфейса для работы с базами данных:

sudo apt-get install libdbi-perl


1.6 Драйвер для работы с базами данных:

sudo apt-get install libdbd-sqlite3-perl


2. Скачать скрипт с GitHub:


https://github.com/GouveaHeitor/nipe


3. Установка NIPE:

cd nipe

chmod +x setup.sh

./setup.sh

perl nipe.pl install


Можно сделать пробный запуск


cd nipe

perl nipe.pl start

perl nipe.pl status


4. Обход ограничений

- В терминале вводим команду:

sudo iptables -t mangle -A POSTROUTING -j TTL --ttl-set 65

- cd nipe


- Подключаем ноутбук к сети (wi-fi, модем и прочее)

- Быстро вводим команды в терминал:

perl nipe.pl start

perl nipe.pl status


После ввода последней команды должен отобразиться изменненный IP-адрес



5. Используем Интернет Yota без ограничений.
