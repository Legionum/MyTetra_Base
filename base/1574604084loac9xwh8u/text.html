Устновка Unreal Engine 4 в системе Linux (на примере Ubuntu 18.04)


Примечание: можно скачивать все с GitHub и без своей учетной записи, но исходный код UnrealEngine хранится в закрытом репозитории, т.е. требуется учетная запись на GitHub, связанная с аккаунтом на EpicGames (создатели UnrealEngine).

1) Авторизоваться на GitHub

2) Скачать архив zip с исходным кодом UnrealEngine.

 https://github.com/EpicGames/UnrealEngine

3) Распаковать скачанный архив во вложенную папку (скорее всего получится папка "UnrealEngine")

4) Открыть консоль. Ввести команды (или по очереди скопировать их из данной статьи).

5)  cd UnrealEngine

6) ./Setup.sh

7) ./GenerateProjectFiles.sh

Примечание:

Если команды ./Setup.sh и ./GenerateProjectFiles.sh  не выполняются то нужно сделать эти файлы исполняемыми. Для этого в консоли вводятся следующие команды:

а) chmod +x Setup.sh

б) chmod +x GenerateProjectFiles.sh

8) Если появляется окно с вопросом о регистрации файлов, то нажимаем "да" или "yes"

9) Ввести в консоль команду:

make

10) Ожидать завершения сборки. Может занять несколько часов (в зависимости от оборудования)

 

Запуск

 

Ввести в консоли следующие команды.

cd UnrealEngine

./Engine/Binaries/Linux/UE4Editor -opengl4

При желании можно подготовить и оформить ярлык для панели задач или рабочего стола.

 

Инструменты для android 

 

После того, как проведена установка движка UnrealEngine и сделан первый проверочный запуск.

Выполняем команды в консоли:

cd Engine/Extras/AndroidWorks/Linux

ls - l

chmod +x CodeWorksforAndroid-1R6u1-linux-x64.run

ls - l

./CodeWorksforAndroid-1R6u1-linux-x64.run

 

Примечание:

возможно нужно скачать другой файл. Чем новее установочный файл, тем новее инструменты для андроид.

Ссылка на новый файл (нужна авторизация).

https://developer.nvidia.com/gameworksdownload#?dn=codeworks-for-android-1r7

 

Примечание:

во  время установки инструментов для андроид может возникнуть сбой (отсутствует файл libpng12).

Необходимо выполнить команды в консоли:

 

wget -q -O /tmp/libpng12.deb http://mirrors.kernel.org/ubuntu/pool/main/libp/libpng/libpng12-0_1.2.54-1ubuntu1_amd64.deb \

&& dpkg -i /tmp/libpng12.deb \

&& rm /tmp/libpng12.deb
