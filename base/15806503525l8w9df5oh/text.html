Запуск игр на Linux


1) Установить программу Lutris


2) После установки драйверов Nvidia версии 435xx и пакета prime, запускать программу

Lutris с параметрами командной строки

prime

Можно для этого вбивать в консоли:

prime lutris


3) Аналогично, требуется запускать Steam (версия для Linux)

prime steam


Если в играх от Valve Corp. (например Black Mesa) не отображается меню (можно не видеть надписи, но слышать звук при наведении на кнопку меню), то необходимо удалить ряд шрифтов


/usr/share/fonts/TTF/Vera.ttf
/usr/share/fonts/TTF/VeraBI.ttf
/usr/share/fonts/TTF/VeraBd.ttf
/usr/share/fonts/TTF/VeraIt.ttf
/usr/share/fonts/TTF/VeraMoBI.ttf
/usr/share/fonts/TTF/VeraMoBd.ttf
/usr/share/fonts/TTF/VeraMoIt.ttf
/usr/share/fonts/TTF/VeraMono.ttf
/usr/share/fonts/TTF/VeraSe.ttf
/usr/share/fonts/TTF/VeraSeBd.ttf


После удаления этих шрифтов меню должно отображаться нормально.


4) Запуск Age of Wanders: Planetfall (Steam)


Запустить Steam, выбрать Proton 4.11 и установить игру


- открыть консоль и выполнить команды


export STEAM_COMPAT_DATA_PATH=~/.local/share/Steam/steamapps/compatdata/718850

cd ~/.local/share/Steam/steamapps/common/Age\ of\ Wonders\ Planetfall/  

~/.local/share/Steam/steamapps/common/Proton\ 4.11/proton run AowPF.exe


Возможно, если в системе специфичным образом устанавливались драйвера на видеокарту, то нужно будет в начало команды добавить слово prime


prime ~/.local/share/Steam/steamapps/common/Proton\ 4.11/proton run AowPF.exe


В дальнейшем, для запуска игры через консоль нужно будет сначала просто запустить Steam и только после этого выполнять команды в консоли
