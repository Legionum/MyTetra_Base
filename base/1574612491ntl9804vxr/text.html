Установка Pyhton в Windows

  • Скачать файл python_docx‑0.8.10‑py2.py3‑none‑any.whl
  • После установки Anaconda можно зайти в папку C:\programdata\anaconda3\scripts , запустить оттуда командную строку
  • выполнить pip install d:\python_docx‑0.8.10‑py2.py3‑none‑any.whl

Примечание

Обратить внимание на путь к файлу программной библиотеки, в примере он расположен на диске D
