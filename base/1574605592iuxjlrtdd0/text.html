Общий алгоритм установки в Linux:

1) Скачать дистрибутив для Python под название Anaconda (объединяет 1,5 тыс. библиотек):

https://www.anaconda.com/distribution/#download-section

2) Установить Anaconda

- Проверить хеш-сумму:

sha256sum Anaconda3-2019.10-Linux-x86_64.sh

Страница, содержащая актуальную хеш-сумму:

https://docs.anaconda.com/anaconda/install/hashes/

Значение хеш-суммы (на дату 1.12.2019)

46d762284d252e51cd58a8ca6c8adc9da2eadc82c342927b2f66ed011d1d8b53

- Запустить установку:

bash Anaconda3-2019.10-Linux-x86_64.sh

- Отвечать на запросы в терминале:

*Согласиться с лицензией:

Yes

*Указать место установки:

\home\legionum\Development\Programming\Anaconda3

*Инициализировать установленную анаконду:

Yes

3) По завершении установки, закрыть и открыть терминал заново

4) Для проверки вводим каманду:

conda

или

conda list

увидим соотвествующую реакцию ((во втором случае, увидим список библиотек)

5) Запускаем графическое приложение "Anaconda Navigator":

anaconda-navigator


6) Для программирования скриптов для 3д-редактора Blender:

conda install -c kitsune.one python-blender


Примечание:

Обратить внимание на Environments (окружения) в Anaconda. Их может быть несколько, каждое со своими библиотеками





Установить Python последней или предпоследней версии в систему

sudo apt-get install python3.6

или

sudo apt-get install python3.7


Установка Anaconda


Возможные проблемы


1. Manjaro Linux. После установки Anaconda может не запускаться Anaconda-navigator (соотвествующая команда в терминале "anaconda-navigator", без ковычек)

Для решения нужно запустить команду установки Anaconda-navigator, находясь в соотвествующем окружении Anaconda (обычно это базовое окружение с названием "base").


How I solved this issue: 1. Be connected to the internet. 2. Open the Anaconda Prompt (looks like a regular command window). If you installed the .exe in your /name/user/ location you should be fine, if not navigate to it. Then start an environment.

conda info --envs

Then run

conda install -c anaconda anaconda-navigator

Press y when prompted (if prompted). It will being downloading the packages needed.

Then run your newly installed Anaconda Navigator

anaconda-navigator

It should start, and also appear in your regular windows 10 apps list.
