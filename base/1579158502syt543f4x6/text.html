Этап создания файла для меню и копирования необходимых файлов (образов и прочего) на флешку


1) Узнать UUID всех разделов

ls -l /dev/disk/by-uuid


Уникальные метки разделов (15.01.2020):


FA232 E934-6B63

NTFS 15C6F8842A182E6C


2) Создать (или скопировать готовый) файл grub.cfg по пути /boot/grub/


**********************************************************************************

Дополнительные команды для терминала


grub-install --removable --target=x86_64-efi --boot-directory=/mnt/USB/boot --efi-directory=/mnt/USB/ /dev/sdb



grub-install --removable --boot-directory=/mnt/USB/boot --efi-directory=/mnt/USB/ /dev/sdb



grub-install --removable --debug --boot-directory=/mnt/USB/boot --target=x86_64-efi --efi-directory=/mnt/USB/ /dev/sdb


Команда для созадния карты устройств

sudo grub-mkdevicemap


Монтирование iso образа в папку


mount -o loop /home/legio/Win10_1809Oct_Russian_x64.iso /mnt/iso/
