Перемещение объектов мышкой¶

❡Мгновенное перемещение объектов¶

❡

Перед создание данной схемы необходимо:¶

  1. Создать БП класса Actor с именем "SphereMove"¶
  2. Разместить объект класса SphereMove на уровне.¶
  3. Правая кнопка мыши на размещенном объекте. Нажать Add Event. Нажать MouseClicked (неточное название). Смысл данной операции в добавлении события к конкретному экземпляру класса SphereMove.¶
  4. Создать новую переменную. Имя переменной "Target_Sphere"¶
  5. Тип переменной "Actor"¶
  6. Значение по умолчанию это имя какого-либо актора на уровне. В данном примере это сфера, которая является экземпляром класса SphereMove. Имя актора это "SphereMove" (также может быть "SphereMove2" и т.д., в зависимости от количества объектов одного класса)¶

 ❡Используемые ноды:¶

  1. GetPlayerController¶
  2. GetHitResultUnderCursor By Channel¶
  3. Break Hit Result¶
  4. BreakVector¶
  5. MakeVector¶
  6. SetActorLocation¶
