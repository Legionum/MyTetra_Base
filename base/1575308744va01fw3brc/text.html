Модификатор Shrinkwrap позволяет одному объекту повторять форму другого (сжиматься вокруг него). Он перемещает каждую вершину объекта, к которому применен модификатор Shrinkwrap, к вершинам целевого (target), с использованием одного из 3-х доступных методов. Как и большинство модификаторов, он может быть применен к меш-объектам, решеткам, кривым, поверхностям и тексту. До тех пор, пока вы не примените модификатор, геометрия объекта останется неизменной, и все изменения будут лишь проецироваться на поверхность другого объекта.


Режим работы: Любой режим

  • Target
    Цель, вокруг которой необходимо обернуть объект.
  • Vertex Group
    Указывает на группу вершин, к которой будет применяться деформация. Также можно задать вес вершинам от 0 до 1, и чем больше значение веса, тем сильнее вершина будет реагировать на модификатор.
  • Offset
    Расстояние между целевыми вершинами и вершинами объекта, к которому применен модификатор.
  • Subsurf Levels
    Применяет (временно) подразделение Catmull-Clark к модифицированному объекту, при использовании режима Projection.
  • Mode
    Здесь задается метод, который будет использоваться для определения ближайшей точки на поверхности целевого объекта (target), для каждой вершины модифицированного объекта. Изменение данного метода будет добавлять специфические опции для каждого режима.
    • Nearest Surface Point – выбирает ближайшую точку на целевом объекте. Добавляет опцию Keep Above Surface, которая располагает модифицированный объект над поверхностью целевого. Не работает если значение Offset равно 0.
    • Nearest Vertex – выбирает ближайшую вершину на целевом объекте. Отсутствуют дополнительные опции.
    • Projection – проецирует вершины вдоль выбранных осей до тех пор, пока они не коснутся целевого объекта. Вершины, которые не могут быть спроецированы, остаются неизменными. Этот метод является самым трудным в освоении, так как он может иногда дать неожиданные результаты… Он добавляет несколько дополнительных вариантов:
  • Subsurf Limit
    Придел расстояния между оригинальными вершинами и поверхностью. Если расстояние больше, чем лимит, вершины не будут проецироваться.
  • X, Y, Z
    Определяет по которой из осей объект будет проецироваться. Эти параметры могут быть объединены друг с другом.
  • Negative, Positive
    Позволяет выбрать направление проекции относительно нормалей. В случае более, чем одного модификатора Shrinkwrap, отрицательные и положительные направления могут быть объединены.
  • Cull Faces
    Позволяет предотвратить любые проекции на “лицевой стороне” (соответственно “обратной стороне”) цели. “Сторона” определяется направлением нормалей объекта.
  • Auxiliary Target
    Дополнительный объект для проецирования.
